# Информационная система «Учёт рабочего времени» (Timesheet App)

Десктопное приложение на Python для автоматизации учёта отработанного времени сотрудников по проектам/задачам и формирования аналитической отчётности для расчёта заработной платы.


## 📌 Суть проекта
Система решает задачу прозрачного мониторинга трудозатрат в компании. Сотрудники могут фиксировать время работы над задачами в реальном времени с помощью встроенного таймера, а руководители — верифицировать эти данные, управлять структурой штата, проектами и выгружать сводные зарплатные отчёты.

---

## 🛠 Стек технологий
* **Язык программирования:** Python 3.10+
* **Графический интерфейс (GUI):** Tkinter + ttk (используется кроссплатформенная тема `clam`)
* **База данных:** SQLite (с поддержкой каскадного удаления `ON DELETE CASCADE`)
* **Система контроля версий:** Git
* **Линтеры и качество кода:** Flake8
* **CI/CD:** GitHub Actions (автоматический синтаксический анализ кода при push/pull_request)

---

## 💻 Функциональные возможности

### Роль: Сотрудник
* Выбор текущего пользователя и активного проекта из актуальной базы данных.
* Фиксация описания выполняемой задачи.
* Запуск, отображение и остановка фонового таймера в реальном времени.
* Автоматическая отправка записи в базу данных со статусом «На рассмотрении».

### Роль: Руководитель (Панель управления)
* **Верификация времени:** Просмотр списка всех незавершённых записей, утверждение (`✓`) или отклонение (`✗`) часов.
* **Зарплатный отчёт:** Генерация агрегированной сводки по утверждённым часам для каждого сотрудника в разрезе проектов.
* **Управление персоналом (CRUD):** Просмотр, добавление и удаление сотрудников.
* **Управление проектами (CRUD):** Просмотр, добавление, редактирование (переименование) и удаление проектов.
* **Каскадное обновление данных:** Все изменения в справочниках мгновенно отображаются в интерфейсе сотрудника.
